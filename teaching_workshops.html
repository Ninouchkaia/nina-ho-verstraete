Genetic Code & Data Sonification Workshop (2017–2025)

Universités Libres des 4 Saisons (Toulouse), Interhack Montpellier

  • Overview of the genetic code and biomolecules (DNA, RNA, proteins).
  • 3D visualization of proteins using PyMol.
  • Data sonification introduction and SuperCollider-based composition.
  • Musical creation from DNA or protein sequences.

Introduction to Algorithmic Music (2017–2025)

Universités Libres des 4 Saisons (Toulouse), Interhack Montpellier

  • Introduction to SuperCollider and live coding concepts.
  • Real-time music creation using FoxDot (Python-based).
  • Building generative musical structures from code.

Agent-Based Simulation with NetLogo (2025)

Interhack Montpellier

  • Modeling complex systems with simple rules (e.g., Game of Life).
  • Exploration of emergent behaviors through simulation.
  • Hands-on creation of agent-based models with NetLogo.
